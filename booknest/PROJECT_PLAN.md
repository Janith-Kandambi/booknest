# BookNest

Goal:
Build a MERN app for personal reading lists and short book reviews.

Core features:
- User auth
- Add/edit/delete books
- Reading status: want to read / reading / completed
- Rating and short review
- Dashboard
- Responsive UI

Frontend stack:
- React + Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Three.js (hero/background only, not everywhere)

Backend stack:
- Node.js
- Express.js
- MongoDB
- JWT auth

Pages:
- Landing page
- Login / Register
- Dashboard
- My Library
- Add Book
- Book Details
- Profile

MVP first:
- Finish full CRUD + auth first
- Then polish UI
- Then add Three.js effects
